# 2 名词解释

| 名词   | 描述                                                                  |
|------|---------------------------------------------------------------------|
| PC模式 | 未接示教器或示教器切换自动模式，由外部电脑中Compass或SDK发送心跳接管控制权限时，为PC模式，此时控制柜PC灯亮起 |
| jog  | 手动控制机器人以时断时续的方式进行移动                                                 |
