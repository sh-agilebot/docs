# 4.14 订阅机器人状态

[#3.1]: ../3-struct/#3-1

[#3.21]: ../3-struct/#3-21

## 4.14.1 订阅

| 方法名  | **subscribe**(`topic_list`: List\[HWState] = None, `io_topic`: List\[Tuple\[IOTopic, int]] = None) -> StatusCodeEnum       |
|------|------------------------------------------------------------------|
| 描述   | 开启订阅机器人状态  支持订阅的机器人状态见[HWState][#3.21]                |
| 请求参数 | `topic_list`: [HWState][#3.21] 默认订阅(机器人JOINT关节值、笛卡尔TCP位置、TF、UF、机器人运行状态、机器人运行速度)  `io_topic`: 自定义需要订阅的IO信号列表 |
| 返回值  | [StatusCodeEnum][#3.1]: 函数执行结果         |
| 兼容的机器人软件版本 | 协作(Copper): v7.5.0.0+  工业(Bronze): v7.5.0.0+    |

## 4.14.2 接收

| 方法名  | **recv**() -> Union\[Dict\[str, Any], None]       |
|------|------------------------------------------------------------------|
| 描述   | 接收机器人消息                        |
| 请求参数 | 无参数         |
| 返回值  | Dict\[str, Any]: 消息内容         |
| 兼容的机器人软件版本 | 协作(Copper): v7.5.0.0+  工业(Bronze): v7.5.0.0+    |

## 4.14.3 取消订阅

| 方法名  | **unsubscribe**() -> StatusCodeEnum       |
|------|------------------------------------------------------------------|
| 描述   | 接收机器人消息                        |
| 请求参数 | 无参数         |
| 返回值  | [StatusCodeEnum][#3.1]: 函数执行结果    |
| 兼容的机器人软件版本 | 协作(Copper): v7.5.0.0+  工业(Bronze): v7.5.0.0+    |
