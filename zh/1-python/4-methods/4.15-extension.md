# 4.15 插件管理

## 4.15.1 获取机器人的IP地址

| 方法名  | **extension.get\_robot\_ip**() -> str    |
|------|------------------------|
| 描述   | 用于在插件环境中获取机器人的IP地址             |
| 请求参数 | 无参数                               |
| 返回值  | str: IP地址 (可能会返回None)                           |
| 机器人软件版本 | 协作机器人: 7.5.0.0  工业机器人: 7.5.0.0    |

示例代码

```python
#!python
from Agilebot.IR.A.extension import Extension

# 初始化插件
extension = Extension()

# 获取IP
res = extension.get_robot_ip()
assert robot_ip == None


```
