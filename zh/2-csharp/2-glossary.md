# 2 名词解释

| 名词    | 描述                                                                                             |
| ----- | ---------------------------------------------------------------------------------------------- |
| PC模式  | 当机器人未连接到示教器，或者示教器未切换到自动模式时，外部计算机通过Compass或SDK发送心跳信号以接管控制权限，此时机器人处于PC模式，并且机器人机柜中的PC指示灯会亮起 |
| 点动    | 以时间间隔的方式手动控制机器人                                                                                |
| 示教器   | 连接在机器人上的手持设备，用于对机器人进行示教和控制                                                                     |
| SDK   | 软件开发工具包，用于对机器人进行编程和控制                                                                          |
| 机器人   | 机器人本身                                                                                          |
| 机器人网络 | 机器人与外部计算机之间的网络连接                                                                               |
