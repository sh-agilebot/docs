请求参数无

工业机器人: 7.5.0.0

示例代码

cs
using Agilebot.IR;
using Agilebot.IR.Types;

public class Test
{
    public static async Task Main()
    {
        string controllerIP = "10.27.1.254";
        
        // 初始化捷勃特机器人
        Arm controller = new Arm(controllerIP);
        // 连接捷勃特机器人
        StatusCode code = await controller.Connect();

        if (code == StatusCode.OK)
        {
            Console.WriteLine($"OVC = {res}");
        }

        if (code == StatusCode.OK)
        {
            Console.WriteLine($"TCSType = {(TCSType)type}");
        }

        // 关闭连接
        code = controller.Disconnect();
        Console.WriteLine(code != StatusCode.OK ? code.GetDescription() : "Successfully disconnected.");
        
    }
}

4.2.16 设置订阅参数 ​

方法名Motion.SetUDPFeedbackParams( bool flag , string ip , int interval , int feedbackType , List<int> DOList = null )
描述配置机器人向指定 IP 地址推送数据的订阅参数
请求参数flag ：bool 是否开启 UDP 数据推送；
ip ：string 接收端的 IP 地址；
interval ：int 发送数据的间隔（单位：毫秒）；
feedbackType ：int 反馈数据格式（0：XML 格式）；
DOList ：list<int> 期望获取的 DO 信号列表（最多十个，可选）
返回值StatusCode: 函数执行结果
备注参数设置仅在 UDP 数据推送功能启用时有效
机器人软件版本协作机器人: 7.5.2.0

示例代码

cs
using Agilebot.IR;
using Agilebot.IR.Types;
using Agilebot.IR.Motion;

public class Test
{
    public static async Task Main()
    {
        string controllerIP = "10.27.1.254";
        
        // 初始化捷勃特机器人
        Arm controller = new Arm(controllerIP);
        // 连接捷勃特机器人
        StatusCode code = await controller.Connect();
        Console.WriteLine(code != StatusCode.OK ? code.GetDescription() : "Successfully connected.");

        // 获取机器人的当前位姿
        code = controller.Motion.SetUDPFeedbackParams(true, "192.168.1.1", 10, 0);
        if (code == StatusCode.OK)
        {
            Console.WriteLine("Successfully set UDP feedback params.");
        }

        // 进入实时位置控制模式
        code = controller.Motion.EnterPositionControl();
        Console.WriteLine(code != StatusCode.OK ? code.GetDescription() : "Successfully entered position control mode.");

        // 在此插入发送UDP数据控制机器人代码

        // 退出实时位置控制模式
        code = controller.Motion.ExitPositionControl();
        Console.WriteLine(code != StatusCode.OK ? code.GetDescription() : "Successfully exited position control mode.");

        // 关闭连接
        code = controller.Disconnect();
        Console.WriteLine(code != StatusCode.OK ? code.GetDescription() : "Successfully disconnected.");
    }
}

4.2.19.15 负载测定全流程 ​

方法名Motion.Payload.PayloadIdentify( double weight = -1, double angle = 90 )
描述完整的负载测定流程，包含上述负载测定全部接口，无特殊需求负载测定只用该接口即可
请求参数weight : double 负载重量（未知重量填 - 1）
angle : double 6 轴允许转动的角度（30-90 度）
返回值返回负载测定结果 PayloadInfo
StatusCode: 函数执行结果
备注返回的负载可以新增到机器人中或写入机器人中已有的某个负载
全流程步骤：
1. 进入负载测定状态
2. 开始负载测定
3. 获取负载测定结果
4. 结束负载测定状态
机器人软件版本协作机器人: 7.5.2.0

示例代码

cs
using Agilebot.IR;
using Agilebot.IR.Motion;
using Agilebot.IR.Types;

public class Test
{
    public static async Task Main()
    {
        string controllerIP = "10.27.1.254";
        
        // 初始化捷勃特机器人
        Arm controller = new Arm(controllerIP);

        // 连接捷勃特机器人
        StatusCode code = await controller.Connect();
        Console.WriteLine(code != StatusCode.OK ? code.GetDescription() : "Successfully connected.");

        // 执行负载测定
        (PayloadInfo payload, code) = controller.Motion.Payload.PayloadIdentify(weight, angle);

        payload.data.id = 6;

        // 保存负载到机器人中
        code = controller.Motion.Payload.AddPayload(payload);

        // 关闭连接
        code = controller.Disconnect();
        Console.WriteLine(code != StatusCode.OK ? code.GetDescription() : "Successfully disconnected.");
    }
}
