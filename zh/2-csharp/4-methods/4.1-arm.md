## 4.1.2 判断与机械臂的连接是否有效

| 方法名  | **IsConnect**()                        |
| ---- | -------------------------------------- |
| 描述   | 判断与机械臂的连接是否有效                |
| 请求参数 | 无参数                               |
| 返回值   | bool: 连接状态，true: 连接有效，false: 连接失效    |
| 兼容的机器人软件版本 | 协作(Copper): v7.5.0.0+  工业(Bronze): v7.5.0.0+    |

## 4.1.3 与机器人断开连接

| 方法名  | **Disconnect**() |
| ---- | ---------------- |
| 描述   | 断开和捷勃特机器人的连接        |
| 请求参数 | 无参数              |
| 返回值   | [StatusCode][#3.1]: 函数执行结果    |
| 兼容的机器人软件版本 | 协作(Copper): v7.5.0.0+  工业(Bronze): v7.5.0.0+    |

示例代码

```cs
using Agilebot.IR;

public class Test
{
    public static async Task Main()
    {
        string controllerIP = "10.27.1.254";

        // 初始化捷勃特机器人
        Arm controller = new Arm(controllerIP);

        // 连接捷勃特机器人
        StatusCode code = await controller.Connect();
        Console.WriteLine(code != StatusCode.OK ? code.GetDescription() : "Successfully connected.");

        // 检查连接状态
        var state = controller.IsConnect();
        Console.WriteLine("Connected: " + state);

        // 关闭连接
        code = controller.Disconnect();
        Console.WriteLine(code != StatusCode.OK ? code.GetDescription() : "Successfully disconnected.");
    }
}
```

## 4.1.12 让机器人伺服重置

| 方法名  | **ServoReset**() |
| ---- | ---------------- |
| 描述   | 当前捷勃特机器人伺服重置         |
| 请求参数 | 无参数              |
| 返回值   | [StatusCode][#3.1]: 函数执行结果 |
| 兼容的机器人软件版本 | 协作(Copper): v7.5.0.0+  工业(Bronze): v7.5.0.0+    |

示例代码

```cs
using Agilebot.IR;
using System.Threading;

public class Test
{
    public static async Task Main()
    {
        string controllerIP = "10.27.1.254";
        
        // 初始化捷勃特机器人
        Arm controller = new Arm(controllerIP);
        // 连接捷勃特机器人
        StatusCode code = await controller.Connect();
        Console.WriteLine(code != StatusCode.OK ? code.GetDescription() : "Successfully connected.");

        // 机械臂伺服重置
        code = controller.ServoReset();
        Console.WriteLine(code != StatusCode.OK ? code.GetDescription() : "Servo Reset.");
        
        // 等待5秒（5000毫秒）
        Thread.Sleep(5000);

        // 机械臂伺服关闭
        code = controller.ServoOff();
        Console.WriteLine(code != StatusCode.OK ? code.GetDescription() : "Servo Off.");

        // 等待5秒（5000毫秒）
        Thread.Sleep(5000); 

        // 机械臂伺服启动
        code = controller.ServoOn();
        Console.WriteLine(code != StatusCode.OK ? code.GetDescription() : "Servo On.");

        // 关闭连接
        code = controller.Disconnect();
        Console.WriteLine(code != StatusCode.OK ? code.GetDescription() : "Successfully disconnected.");
    }
}
```
