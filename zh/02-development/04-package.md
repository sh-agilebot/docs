# 打包与安装

插件开发完毕后，可以使用我们提供的工具`捷勃特插件打包工具`进行打包成特定的格式。

[软件下载链接](https://github.com/sh-agilebot/extension-toolkit/releases)

软件界面如图所示：

![](/images/development/插件打包工具.png)

## 打包插件为.gbtapp格式

1. 点击选择目录按钮，选择开发好的插件目录。

![](/images/development/插件打包1.png)

2. 根据提示填写参数，并点击打包插件按钮，工具会在当前目录下生成插件包。

![](/images/development/插件打包2.png)

![](/images/development/插件打包3.png)

## 安装插件

在AgileLink中的插件管理页面安装插件。

![](/images/development/插件安装1.png)

:::warning
在Bronze中，无法在示教器上选择文件，请通过PC安装插件包。
:::
