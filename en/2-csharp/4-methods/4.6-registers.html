4.6 Register Information ​

4.6.1 R Numeric Register Operations ​

4.6.1.1 Reading the Value of an R Register ​

Method NameRegisters.Read_R(int index )
DescriptionReads the value of an R register.
Request Parametersindex : The index of the register to read
Return Valuedouble: Register value
StatusCode: Result of function execution
Compatible robot software versionCollaborative (Copper): v7.6.0.1+
Industrial (Bronze): v7.6.0.0+

4.6.1.2 Writing the Value of an R Register ​

Method NameRegisters.Write_R(int index , double value )
DescriptionWrites the value of an R register.
Request Parametersindex : int The index of the register to write
value : double The value to write to the R register
Return ValueStatusCode: Result of function execution
Compatible robot software versionCollaborative (Copper): v7.6.0.1+
Industrial (Bronze): v7.6.0.0+

4.6.1.3 Deleting an R Register ​

Method NameRegisters.Delete_R(int index )
DescriptionDeletes an R register.
Request Parametersindex : The index of the register to delete
Return ValueStatusCode: Result of function execution
Compatible robot software versionCollaborative (Copper): v7.5.0.0+
Industrial (Bronze): v7.5.0.0+

Example Code

cs
using Agilebot.IR;

public class Test
{
    public static async Task Main()
    {
        string controllerIP = "10.27.1.254";

        // Initialize the Agilebot robot
        Arm controller = new Arm(controllerIP);
        // Connect to the Agilebot robot
        StatusCode code = await controller.Connect();
        Console.WriteLine(code != StatusCode.OK ? code.GetDescription() : "Successfully connected.");

        int index = 1;

        // Register operations
        code = controller.Registers.Write_R(index, 9.9);
        Console.WriteLine(code != StatusCode.OK ? code.GetDescription() : "Successfully write.");
        (double value, code) = controller.Registers.Read_R(index);
        Console.WriteLine(code != StatusCode.OK ? code.GetDescription() : $"Value: {value}");
        code = controller.Registers.Delete_R(index);
        Console.WriteLine(code != StatusCode.OK ? code.GetDescription() : "Successfully delete.");

        if (code == StatusCode.OK)
        {
            Console.WriteLine($"Success");
        }

        // Close the connection
        code = controller.Disconnect();
        Console.WriteLine(code != StatusCode.OK ? code.GetDescription() : "Successfully disconnected.");
    }
}
