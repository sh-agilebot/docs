3.1 StatusCodeEnum ​

Description ​

Return status codes for interfaces.

Note: If a return code not listed in the documentation is encountered, refer to the robot fault code table or interpret based on the literal meaning of the return value.

python
from Agilebot.IR.A.status_code import StatusCodeEnum

3.33 SignalType ​

Description ​

SignalType is an enumeration class used to distinguish different types of input and output signals. It covers digital signals, special signals, arm signals, group signals, analog signals, and more, helping to accurately identify and manage different types of signals in the robot control system.

Import ​

python
from Agilebot.IR.Types import SignalType

Enum Values ​

NameEnum ValueDescription
DI1Digital Input signal
DO2Digital Output signal
UI3User Input signal
UO4User Output signal
RI5Robot Input signal
RO6Robot Output signal
GI7Group Input signal
GO8Group Output signal
TAI9Tool Analog Input signal
TDI10Tool Digital Input signal
TDO11Tool Digital Output signal
AI12Analog Input signal
AO13Analog Output signal

3.33.1 SignalValue ​

Description ​

The SignalValue class is used to define the on/off values of signals. It uses two constants, OFF and ON , to represent the off and on states of signals. This class is used in the robot control system to control and read the state of signals.

Import ​

python
from Agilebot.IR.Types import SignalValue

Attributes ​

NameValueDescription
OFF0Signal off
ON1Signal on
