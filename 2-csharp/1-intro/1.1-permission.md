# 1.1 操作权限

使用SDK必须在上位机获得操作权限的状态下。

若IRC-I4A-C系列控制柜上插有示教器，必须将示教器切换到自动模式，上位机能够连接上控制柜；
上位机拥有操作权限后，示教器失去操作权限，此时PC状态灯会亮绿色，TP状态指示灯不亮；若示教器为手动模式，则无法获得权限，示教器具有操作权限。

若IRC-I4A-C系列控制柜上没插示教器，则可通过上位机直接连接IRC-I4A-C系列控制柜获得操作权限。

| 系统状态指示灯及按钮  | 功能                                         |
| ----------- | ------------------------------------------ |
| TP状态灯       | 控制柜插入示教器时，该灯亮绿色，否则不亮                       |
| PC状态灯       | 使用虚拟示教器软件Compass进行操作，该灯亮绿色，否则不亮            |
| AUTO状态灯     | 机器人的运行模式为自动运行时或示教器和Compass无控制权时，该灯亮绿色，否则不亮 |
| TP BYPASS按钮 | 拔出示教器前，若机器人是手动模式，拔出示教器后可常按此按钮将机器人切换到自动模式   |
