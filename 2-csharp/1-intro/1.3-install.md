# 1.3 安装

### IDE安装

* 推荐使用Visual Studio作为开发环境

* Visual Studio下载地址：[下载 Visual Studio Tools - 免费安装 Windows、Mac、Linux](https://visualstudio.microsoft.com/zh-hans/downloads/)

* 下载并安装好Visual Studio后，启动软件

### SDK安装

* 新建C#控制台项目，框架选择 .NET 6.0 或更高
  ![示例图片](./images/1.jpg)
  ![示例图片](./images/2.jpg)

* 点击项目菜单中的属性，修改目标OS为Windows， 目标操作系统版本为 7.0 或更高
  ![示例图片](./images/8.jpg)
  ![示例图片](./images/9.jpg)

* 点击工具菜单中的NuGet包管理器，右上角程序包源处点击设置按钮，添加本SDK包所在目录为程序包源
  ![示例图片](./images/3.jpg)
  ![示例图片](./images/4.jpg)
  ![示例图片](./images/5.jpg)
  ![示例图片](./images/6.jpg)

* 点击程序包源切换为新添加的源，安装Agilebot.SDK包
  ![示例图片](./images/7.jpg)

* 开始编写代码，运行时上位机必须连接到机器人网络中或与机器人处于同一局域网下
  ![示例图片](./images/10.jpg)
  ![示例图片](./images/11.jpg)
